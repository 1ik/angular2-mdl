Tabs components

These are standalone tabs that are not included in the basic layouts. Code a mdl-tabs element and within this component add mdl-tab-panel components. Don't be confused with the mdl-layout-tab-panel.

Attributes of the mdl-tabs component
Attribute Description
mdl-tab-active-index Secifies the index of the active tab.
mdl-tab-active-changed Event-listener function that will be fired if the activ tab index changed. The changed index is stored in the index property of the event object.
mdl-ripple BooleanPorperty. If present or true the tabs will ripple on click or tap.
Attributes of the mdl-tab-panel component
Attribute Description
mdl-tab-panel-title The content of this attribute is used to create the tab title.
